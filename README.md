bartnodeapi
===========

BART JSON API using Node and Express.  Will listen on port 8888, supports JSONP.

Status
------

* Right now, this is an API facade
* Responses are hard coded

Installation
------------

* Install node
* Make sure node is on your path

Building
--------

Get depenencies:

```
npm install
```

Running
-------

Start:

```
node server.js
```
